FDSN Data Downloader

Search for seismic stations and download waveforms and StationXML metadata from 25+ global FDSN-compliant web services.

Overview

The FDSN (International Federation of Digital Seismograph Networks) Downloader allows you to discover seismic stations near a reference earthquake and download the corresponding waveform data (MiniSEED) and instrument response files (StationXML) in one workflow, without leaving SeisBox.

Access from: Data → FDSN Data Downloader. An active internet connection is required.

Supported Data Providers

SeisBox pre-configures 25 major FDSN nodes. Select one or multiple providers using the checkboxes at the top of the left panel:

Station Search Parameters

ParameterDefaultDescription
Min Radius (degrees)0.0Inner radius of the annular search zone around the epicenter. Useful to exclude very close stations.
Max Radius (degrees)5.0Outer radius of the search zone. 1° ≈ 111 km.
ChannelBHZ,HHZComma-separated list of channel codes to search for. Wildcards supported: BH? matches BHZ, BHN, BHE.

Time Window

ParameterDefaultDescription
Start Offset (s)−60Start of the waveform window relative to origin time (negative = before the earthquake).

Workflow

  1. Select Providers

    Check the FDSN nodes you want to query. For teleseismic data, IRIS is usually sufficient. For regional networks, add the appropriate regional provider.

  2. Enter Reference Earthquake

    Fill in the epicenter coordinates and origin time. You can copy these from the ISC Catalog Search results.

  3. Click "🔍 Search Stations"

    Searches all selected providers simultaneously. Found stations appear in the table at the bottom and as blue diamonds on the map.

  4. Download Waveforms

    Click "⬇️ Download Waveforms" and choose an output folder. All found stations are downloaded as MiniSEED files. Progress is shown with a percentage bar.

  5. Download StationXML (Optional)

    Click "📥 Download StationXML" to also download instrument response files in StationXML format (needed for deconvolution/instrument correction). A stations.csv summary is also saved.

After downloading, use File → Open Folder in the Waveform Viewer to load all MiniSEED files from the output folder at once.

Station Table

The results table at the bottom shows all found stations with: Provider, Network code, Station code, Latitude, Longitude, Elevation (m), and Site Name. The table is sortable and can be exported as part of the StationXML download workflow.

Map Display

The right panel shows a global map with:

  • Red asterisk: The reference earthquake epicenter.
  • Blue circle: Inner radius boundary.
  • Green circle: Outer radius boundary.
  • Blue diamonds: Found seismic stations.
