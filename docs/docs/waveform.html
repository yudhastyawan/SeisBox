Waveform Viewer

Load, inspect, and navigate multi-component seismic waveforms in SAC and MiniSEED formats with full phase picking support.

Overview

The Waveform Viewer is the central workspace of SeisBox. It displays seismic traces in a multi-panel time-series view, supports interactive zoom/cut operations, bandpass filtering, spectrogram computation, and phase annotation. Picks are saved automatically to companion .picks files alongside the original data file.

Loading Data

Go to File → Open File(s) to load individual SAC or MiniSEED files, or File → Open Folder to load all waveform files in a directory. SeisBox automatically groups traces by station-component and populates the sidebar.

Supported formats: SAC (.sac, .SAC) and MiniSEED (.mseed, .seed).

Navigation

  • Click a trace in the sidebar to bring it into focus.
  • Use Shift+, / Shift+. to navigate to the previous/next station group.
  • Scroll the mouse wheel over the plot to pan; hold Ctrl to zoom.
  • Drag to pan the view.

Phase Picking

Phase picks are placed by hovering the mouse over the time position on any trace and pressing the corresponding key. Picks are shown as vertical colored lines on the plot.

KeyAction
pPlace / update P-wave start pick at cursor position
oPlace / update P-wave end pick
sPlace / update S-wave start pick
aPlace / update S-wave end pick
Shift+PDelete P-start pick
Shift+SDelete S-start pick
Shift+ODelete P-end pick
Shift+ADelete S-end pick
wWipe all picks on the active trace

Pick Metadata

After placing a pick, hover close to it and press the following keys to annotate it:

KeyAnnotation
iSet onset to Impulsive
eSet onset to Emergent
uSet first-motion polarity to Up
dSet first-motion polarity to Down
[Decrease pick uncertainty by 0.01 s
]Increase pick uncertainty by 0.01 s
=Capture amplitude at cursor position for the nearest pick

View Controls

KeyAction
c (twice)Cut the X-axis view to a time range (press once to set start, again for end)
Shift+CReset the cut — restore full trace view
z (twice)Zoom the X-axis to a selected time range
Shift+ZReset zoom to full view

Filtering

KeyAction
bOpen/close the Bandpass Filter dialog
fToggle Predictive Filter (requires a P-start pick)
Shift+B or Shift+FRemove all filters — restore original trace

Analysis Tools

KeyAction
qCompute and display Spectrogram for the active trace (within current view)
Shift+QHide spectrogram
hToggle Hodogram display

For large waveforms (>100,000 samples), SeisBox will ask for confirmation before computing the spectrogram to avoid performance issues.

Automatic Pick Saving

Every time a pick is placed, modified, or deleted, SeisBox auto-saves the pick set to a .picks file next to the original seismic file. The file format is a simple tab-delimited text file compatible with common seismic analysis workflows. Picks are automatically loaded the next time you open the same file.

Exporting Picks

Go to File → Export Picks to save all picks to a structured text file. Use File → Export Image to save a PNG of the current trace view.
