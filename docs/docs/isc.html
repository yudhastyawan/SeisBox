ISC Catalog Search

Search and download earthquake catalogs from the International Seismological Centre (ISC) directly within SeisBox, with an interactive map for bounding box selection.

Overview

The ISC Catalog Search module connects to the ISC Bulletin API to retrieve earthquake event data (time, location, depth, magnitude) for a user-defined geographic region and time window. Downloaded events are displayed on the built-in map and can be passed to the Spatial Analysis, BVor Analysis, or Coulomb Stress modules.

Access from: Catalog → ISC Catalog Search (or the spatial analysis toolbar). An active internet connection is required.

Geographic Bounding Box

Define the search area using the interactive map:

  1. Click "Draw BBox"

    The cursor changes to bounding-box draw mode.

  2. Click and drag on the map

    Draw a rectangle over the region of interest. The coordinates are shown next to the button and sent to the ISC API.

  3. Adjust

    Click "Draw BBox" again to redraw a new rectangle.

You can also pan and zoom the map before drawing. Use the mouse wheel to zoom, and click-drag to pan (when not in Draw BBox mode).

Searching the Catalog

Click "Search ISC" to initiate the query. The search runs asynchronously in the background — the button label changes to "Searching..." and a status message at the bottom of the window shows progress. Large queries (many events or long time windows) may take 10–60 seconds.

The number of returned events is displayed in the status bar after completion. Events are plotted as dots on the map, sized and colored by magnitude.

Cross-Section View

After loading events, you can define a cross-section profile on the map to visualize seismicity along a line (e.g., along a fault trace). Depth-distance cross-section plots are available in the Spatial Analysis view.

Exporting the Catalog

  • CSV Export: Use File → Export Catalog CSV to save all loaded events as a CSV file with columns: longitude, latitude, depth, magnitude, time.
  • Raw ISC Data: The raw ISC bulletin text is stored internally and can be saved via File → Save Raw ISC Data.

Using Events in Other Modules

Events loaded from ISC are automatically available to other analysis modules:

  • BVor Analysis: Leave the CSV field empty — the module will use the active ISC events directly.
  • Spatial Analysis: The map and cross-section views reflect the currently loaded events.
  • Coulomb Stress Batch: Export the catalog as CSV, then use it as the batch receiver file.
