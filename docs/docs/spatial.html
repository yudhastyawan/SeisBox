Spatial Analysis

Map-based earthquake catalog visualization with cross-section projection, magnitude-time plots, and tectonic plate boundary overlay.

Overview

The Spatial Analysis view is integrated within the ISC Catalog Search window. It provides a geographic map of the loaded earthquake catalog with tools for filtering, cross-section profiling, and statistical visualization.

Access from: Catalog → ISC Catalog Search. The map is the right panel of the combined ISC/Spatial window.

Map Display

Events are plotted as circles on the map. The map includes:

  • Coastlines: Natural Earth 110m land polygons (bundled with the application — no internet required for the base map).
  • Tectonic Plate Boundaries: PB2002 plate boundary dataset (Bird, 2003) displayed as red/orange lines.
  • Earthquake Events: Circles sized proportionally to magnitude and colored by depth.

Map Interaction Modes

  • Default (Pan & Zoom): Scroll to zoom, click-drag to pan.
  • Draw BBox: Click "Draw BBox" to enter bounding box draw mode for the ISC search area. Click and drag to define the rectangle, then release.
  • Cross-Section Line: Click "Draw Profile" to draw a cross-section line. Click two points on the map to define the profile start and end.

Magnitude-Time Plot

Below the map, a magnitude vs. time scatter plot is displayed for all loaded events. This allows quick visual inspection of:

  • Temporal clustering of seismicity (aftershock sequences)
  • Completeness of the catalog (minimum detectable magnitude over time)
  • Main shock and aftershock relationships

Cross-Section Analysis

Define a profile line on the map to project earthquake hypocenters onto a vertical cross-section plane. This is useful for:

  • Visualizing the geometry of a fault plane in depth
  • Identifying seismically active structures and their dip angles
  • Differentiating crustal from mantle seismicity

The cross-section plot shows distance along the profile (horizontal axis) vs. depth (vertical axis, increasing downward). Events within a defined corridor width of the profile line are projected perpendicularly onto the section plane.

Cross-Section Parameters

ParameterDescription
Profile Start / EndTwo points defining the section line on the map (set by clicking in Draw Profile mode).
Corridor Width (km)Events within this perpendicular distance from the profile line are included in the cross-section.
Max Depth (km)Depth range displayed on the cross-section.

Filtering Events

The event list can be filtered using the search/filter controls in the left panel:

  • Magnitude range (Min/Max Mag)
  • Depth range (Min/Max Depth)
  • Date/time range
  • Geographic bounding box (from map selection)

Filtered events are immediately reflected in all views (map, magnitude-time plot, cross-section).

Exporting

  • CSV: Export the current filtered event list as a CSV file for use in other tools.
  • Processed CSV: Saves a cleaned version of the ISC catalog with standardized column names.
  • Map as Image: Right-click the map and select "Save as PNG" to export the current map view.

Integration with Other Modules

Events loaded in this window are shared with:

  • BVor Analysis: Uses the spatial event list for b-value mapping.
  • Coulomb Stress (Batch): Export the catalog to use individual focal mechanisms as receiver faults.
