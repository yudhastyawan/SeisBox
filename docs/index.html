Seismic Analysis
In One Box.

SeisBox is a fast, native macOS desktop application for seismic data analysis. Load waveforms, pick phases, compute HVSR, model Coulomb stress, and explore earthquake catalogs — all in one place.

Everything you need for seismic analysis

A comprehensive toolkit designed for researchers and practitioners.

🌊

Waveform Viewer

Load and visualize SAC and MiniSEED seismic waveforms with interactive zoom and multi-trace display.

📌

Phase Picking

Interactively pick P- and S-wave arrivals directly on seismograms with precision tools.

📊

HVSR Analysis

Compute Horizontal-to-Vertical Spectral Ratios (HVSR) with full SESAME criteria evaluation.

🔬

RJ-MCMC Inversion

Bayesian Reversible-Jump MCMC inversion for S-wave velocity structure from HVSR curves.

⚡

Coulomb Stress

Compute Coulomb Failure Stress changes using the Okada dislocation model with fault optimization.

🗺️

Spatial Analysis

Map-based earthquake catalog visualization with cross-section tools and magnitude-time plots.

🌐

ISC Catalog Search

Download and filter earthquake catalogs directly from the International Seismological Centre.

📡

FDSN Downloader

Download seismic waveforms from any FDSN-compliant web service with station and channel filters.

🔷

BVor Analysis

Bayesian Voronoi spatial analysis for detecting seismicity rate changes in time and space.

Get Started

Up and running in under a minute.

1

Download

Get the latest SeisBox_macOS.zip from the GitHub Releases page.

⬇ Latest Release
2

Extract & Fix Permissions

Extract the zip. Then double-click Fix_App_First.command to remove the macOS quarantine flag (required once).

3

Launch SeisBox

Double-click SeisBox.app and start analyzing your seismic data!

Build from Source

git clone git@github.com:yudhastyawan/SeisBox.git
cd SeisBox
chmod +x compile_dist.sh
./compile_dist.sh

Requires Rust 1.70+.
